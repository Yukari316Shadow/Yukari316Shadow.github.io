给我整一个色图

这是一个突然一时兴起而写的一个API，通过这个API可以产生随机的色图

一些废话（

所有的图片均为我和我的朋友（我真有朋友）进行挑选并上传进行维护，并不是爬的嗷，所以图片都是好康的图片（而且都和我的xp有关系，当然也和我朋友有关系）

看看域名是什么（

所有的图片均来自Pixiv，图片的所属版权均为画师所有

本API只提供随机色图的功能，并不提供搜索/代理等功能

在这个图库的所有图片不一定很色，但一定都是让我康康！的（这也叫色图？.jpg）

使用本API产生的所有问题都都与我无关，不提供任何数据库源数据，请节制使用

API

GET https://api.yukari.one/setu/

API Key

注意
以后可能会在Discord或者是Telegram实装API Key的申请以限制API的调用次数

Request

字段名 必须 默认 说明
r18 否 0 0为正常图片,1为R18图片

Response

响应类型:JSON

字段名 类型 说明
code int 0成功,500服务器内部错误
message string 返回信息
count int 图片数量
data array 包含图片信息的数组

data数组

字段名 类型 说明
pid int 图片PID[Pixiv]
uid int 画师UID[Pixiv]
title string 图片标题
author string 画师名字
r18 bool 是否是R18图片
tags array 标签数组
url string 图片地址[pixiv.cat proxy]

有些图片不是R18却带着R18的tag
